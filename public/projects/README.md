# Imágenes de Proyectos

Coloca las imágenes de cada proyecto en sus respectivas carpetas:

## Estructura de carpetas:

```
public/
  projects/
    matchear/
      1.jpg (o .png, .webp)
      2.jpg
      3.jpg
      ...
    
    dashboard-deportivo/
      1.jpg
      2.jpg
      3.jpg
      ...
    
    fobibike/
      1.jpg
      2.jpg
      3.jpg
      ...
```

## Instrucciones:

1. **Matchear**: Coloca las imágenes en `public/projects/matchear/`
   - Nombra las imágenes como: `1.jpg`, `2.jpg`, `3.jpg`, etc.

2. **Dashboard Deportivo**: Coloca las imágenes en `public/projects/dashboard-deportivo/`
   - Nombra las imágenes como: `1.jpg`, `2.jpg`, `3.jpg`, etc.

3. **Fobibike**: Coloca las imágenes en `public/projects/fobibike/`
   - Nombra las imágenes como: `1.jpg`, `2.jpg`, `3.jpg`, etc.

## Formatos soportados:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Nota:
Las imágenes se mostrarán en un carrusel con navegación por flechas y puntos indicadores.
